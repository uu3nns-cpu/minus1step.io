# 썸네일 스튜디오 - 블로그 썸네일 제작 도구

다양한 비율(1:1, 9:16, 3:4, 4:3, 16:9)의 블로그 썸네일을 손쉽게 만들 수 있는 웹 기반 도구입니다.

## 🌟 주요 기능

### 📐 다양한 비율 지원
- **1:1** - 인스타그램, 블로그 정사각형 (1000x1000px)
- **9:16** - 인스타그램 스토리, 릴스 (1080x1920px)
- **3:4** - 세로형 SNS 포스트 (1080x1440px)
- **4:3** - 블로그 헤더 (1440x1080px)
- **16:9** - 유튜브 썸네일, 블로그 배너 (1920x1080px)

### 🎨 스타일 프리셋
12가지 전문가가 디자인한 프리셋 스타일:
- **모던** - 보라색 그라디언트
- **그라디언트** - 핑크 그라디언트
- **미니멀** - 심플한 화이트
- **볼드** - 강렬한 검정/노랑
- **오션** - 시원한 블루 그라디언트
- **선셋** - 따뜻한 오렌지 그라디언트
- **네온** - 사이버펑크 스타일
- **파스텔** - 부드러운 파스텔톤
- **빈티지** - 클래식 빈티지
- **다크** - 다크모드 스타일
- **자연** - 그린 그라디언트
- **캔디** - 화려한 핑크/퍼플

### 🎭 텍스트 편집
- **이모지 픽커**: 15가지 인기 이모지 원터치 삽입
- **글자수 카운터**: 실시간 글자수 표시 (최대 200자)
- **자동 줄바꿈**: 긴 텍스트 자동 줄바꿈 처리
- **다양한 폰트**: 20가지 이상의 한글/영문 폰트
  - 산세리프 (라이트, 레귤러, 볼드)
  - 세리프 (라이트, 볼드)
  - 디스플레이 (가벼움, 근육, 굵게)
  - 손글씨

### 🎨 고급 커스터마이징
- **색상 선택**:
  - 배경색, 글자색, 테두리색 독립 제어
  - 16가지 색상 팔레트
  - 컬러 피커로 자유로운 색상 선택
  - HEX 코드 직접 입력
  - **최근 사용 색상**: 최근 사용한 8가지 색상 자동 저장
  
- **글자 크기**: 50% ~ 180% 조절

- **텍스트 테두리**:
  - 테두리 활성화/비활성화
  - 테두리 색상 선택
  - 테두리 두께 조절 (1px ~ 20px)

- **썸네일 테두리**:
  - 캔버스 테두리 추가
  - 테두리 색상 선택
  - 테두리 두께 조절 (5px ~ 100px)

### ⚡ 편의 기능
- **랜덤 스타일**: 🎲 버튼으로 랜덤 스타일 생성
- **초기화**: 🔄 모든 설정을 기본값으로 복원
- **자동 저장**: 모든 설정 자동 저장 (localStorage)
- **실시간 미리보기**: 변경사항 즉시 반영
- **다운로드 피드백**: 다운로드 진행 상태 표시

### 🎯 사용자 경험
- **탭 인터페이스**: 기본 설정 / 고급 설정 분리
- **Glassmorphism 디자인**: 세련된 글래스 효과 UI
- **애니메이션 배경**: 부드러운 그라디언트 배경 애니메이션
- **반응형 디자인**: 모바일, 태블릿, 데스크탑 최적화
- **키보드 접근성**: Tab 키로 모든 컨트롤 접근 가능
- **디바운스 최적화**: 입력 시 성능 최적화

## 🚀 사용 방법

1. **비율 선택**: 원하는 썸네일 비율 선택 (1:1, 9:16, 3:4, 4:3, 16:9)

2. **스타일 선택**:
   - 12가지 프리셋 중 하나 선택
   - 또는 "랜덤" 버튼으로 자동 생성
   - 또는 "고급 설정" 탭에서 직접 커스터마이징

3. **텍스트 입력**:
   - 썸네일에 표시할 텍스트 입력 (최대 200자)
   - 이모지 버튼으로 이모지 추가
   - 엔터키로 줄바꿈 가능

4. **세부 조정** (기본 설정):
   - 글꼴 선택
   - 글자 크기 조절 (50% ~ 180%)

5. **고급 설정** (고급 설정 탭):
   - 배경색, 글자색 세밀 조정
   - 최근 사용 색상에서 빠른 선택
   - 글자 테두리 추가 및 조절
   - 썸네일 테두리 추가 및 조절

6. **다운로드**: 💾 다운로드 버튼 클릭

## 🎨 디자인 특징

### Glassmorphism UI
- **반투명 배경**: backdrop-filter blur 효과
- **애니메이션 배경**: 부드러운 그라디언트 전환
- **글래스 컨테이너**: 모던한 글래스 효과
- **호버 애니메이션**: 직관적인 인터랙션

### 색상 시스템
- **16가지 배경 팔레트**: 전문가가 선별한 배경색
- **16가지 텍스트 팔레트**: 가독성 높은 글자색
- **16가지 테두리 팔레트**: 다양한 테두리 색상
- **최근 사용 색상**: 자동 저장 및 빠른 접근

### 반응형 디자인
- **데스크탑**: 2컬럼 레이아웃 (미리보기 + 컨트롤)
- **태블릿**: 1컬럼 레이아웃
- **모바일**: 최적화된 터치 인터페이스
