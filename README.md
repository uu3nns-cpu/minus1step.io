# 썸네일 스튜디오

다크 미니멀 디자인의 전문적인 블로그 썸네일 제작 도구입니다.

## 📁 프로젝트 구조

```
minus1step.io-main/
├── index.html                  # 메인 HTML 파일
├── dark-minimal-glass.css      # 스타일시트
├── js/                         # JavaScript 모듈
│   ├── config.js              # 데이터 및 설정
│   ├── utils.js               # 유틸리티 함수
│   ├── colors.js              # 색상 관리
│   ├── ui.js                  # UI 컨트롤
│   ├── presets.js             # 프리셋 관리
│   ├── canvas.js              # 썸네일 생성 및 다운로드
│   ├── storage.js             # 설정 저장/불러오기
│   └── init.js                # 초기화 및 이벤트 리스너
├── docs/                       # 문서 파일들
└── _backup/                    # 백업 파일들
```

## ✨ 주요 기능

### 1. 다양한 비율 지원
- 1:1 (정사각형)
- 9:16 (세로형, 스토리)
- 3:4 (세로형)
- 4:3 (가로형)
- 16:9 (와이드)

### 2. 12가지 스타일 프리셋
- 모던 - 보라색 그라디언트
- 그라디언트 - 핑크 그라디언트
- 미니멀 - 깔끔한 흰색
- 볼드 - 강렬한 검정-노랑
- 오션 - 파란색 그라디언트
- 선셋 - 빨강-노랑 그라디언트
- 네온 - 네온 효과
- 파스텔 - 부드러운 파스텔 톤
- 빈티지 - 빈티지 스타일
- 다크 - 다크 모드
- 자연 - 자연 그라디언트
- 캔디 - 화려한 그라디언트

### 3. 고급 설정
- 배경색/글자색 커스터마이징
- 다양한 한글 폰트 (16종)
- 글자 크기 조절 (50-180%)
- 글자 테두리 추가
- 썸네일 테두리 추가
- 최근 사용 색상 기록

### 4. 편의 기능
- 실시간 미리보기
- 이모지 입력
- 랜덤 스타일 생성
- 설정 자동 저장
- 키보드 단축키 (Ctrl+S: 다운로드, Ctrl+R: 랜덤)

## 🚀 최근 개선사항

### 성능 최적화
- **일시저장 빈도 개선**: debounce 시간을 1초로 늘려 불필요한 저장 최소화
- **폰트 로딩 개선**: 여러 크기로 사전 로드하여 일관성 확보
- **비동기 처리**: 모든 렌더링 작업에 async/await 적용

### 버그 수정
- 일부 글자만 폰트가 다르게 보이던 문제 해결
- 스타일 선택이 즉시 적용되지 않던 문제 해결
- 프리셋 적용 시 폰트 weight 정보 손실 문제 해결

### UI 개선
- subtitle 제거로 헤더 간소화
- 제목의 이모지(🎨) 제거
- 더 깔끔한 인터페이스

## 🎨 사용 방법

1. **텍스트 입력**: 썸네일에 표시할 텍스트 입력
2. **비율 선택**: 원하는 썸네일 비율 선택
3. **스타일 선택**: 12가지 프리셋 중 하나 선택 또는 랜덤 버튼 클릭
4. **세부 조정**: 고급 설정에서 색상, 폰트, 크기 등 조정
5. **다운로드**: 완성된 썸네일을 PNG로 다운로드

## 🛠 기술 스택

- **HTML5 Canvas**: 썸네일 렌더링
- **JavaScript ES6+**: 모듈화된 코드 구조
- **CSS3**: 다크 미니멀 글래스모피즘 디자인
- **Google Fonts**: 16종의 한글 폰트
- **LocalStorage**: 설정 자동 저장

## 🔧 커스터마이징

### 새 프리셋 추가
`js/config.js`의 `presets` 객체에 새 프리셋 추가:

```javascript
myPreset: {
    bgColor: '#색상코드',
    textColor: '#색상코드',
    gradient: true,
    gradientColor: '#색상코드',
    fontFamily: "'폰트명', sans-serif",
    fontWeight: '700'
}
```

### 새 폰트 추가
1. `index.html`에 Google Fonts 링크 추가
2. `<select id="fontFamily">`에 옵션 추가
