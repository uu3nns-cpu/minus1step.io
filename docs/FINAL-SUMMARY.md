## 🎨 다크 버전의 특징

### 1. **다크 테마 색상**
```css
/* BlogWriter 다크 팔레트 */
--bg-primary: #121212        /* 깊은 블랙 */
--bg-secondary: #1e1e1e      /* 카드 배경 */
--bg-tertiary: #2d2d2d       /* 버튼 배경 */

--text-primary: #e0e0e0      /* 메인 텍스트 */
--text-secondary: #b0b0b0    /* 부텍스트 */
--text-tertiary: #888888     /* 힌트 */

--accent-primary: #8b7ee8    /* 보라색 강조 */
--border-color: #3d3d3d      /* 테두리 */
```

### 2. **미니멀 디자인**
- ✅ 불필요한 요소 제거
- ✅ 깔끔한 간격
- ✅ 명확한 계층
- ✅ 집중력 향상

### 3. **글래스모피즘 효과**
```css
background: rgba(30, 30, 30, 0.7)
backdrop-filter: blur(20px)
border: 1px solid rgba(255, 255, 255, 0.1)
```

### 4. **부드러운 애니메이션**
- 배경 그라디언트 움직임
- 호버 시 상승 효과
- 부드러운 전환 (0.3s)

## 🖼️ 시각적 비교

### 배경
| 기존 | 다크 버전 |
|------|----------|
| 밝은 보라-핑크 그라디언트 | 다크 네이비-블루 그라디언트 |
| `#667eea → #764ba2` | `#1a1a2e → #16213e` |

### 카드
| 기존 | 다크 버전 |
|------|----------|
| `rgba(255, 255, 255, 0.15)` | `rgba(30, 30, 30, 0.7)` |
| 밝고 투명한 | 어둡고 차분한 |

### 텍스트
| 기존 | 다크 버전 |
|------|----------|
| 흰색 (밝은 배경 위) | `#e0e0e0` (어두운 배경 위) |
| 그림자로 가독성 | 높은 대비로 가독성 |

### 버튼
| 기존 | 다크 버전 |
|------|----------|
| 밝은 그라디언트 | 차분한 단색 + 그라디언트 |
| 화려한 효과 | 세련된 효과 |

## 🎨 디자인 하이라이트

### 배경 애니메이션
```css
/* 다크 버전 배경 */
background: linear-gradient(
    135deg, 
    #1a1a2e 0%,   /* 다크 네이비 */
    #16213e 50%,  /* 미드나이트 블루 */
    #0f1624 100%  /* 깊은 블랙 */
);

/* 부드러운 오버레이 */
radial-gradient(
    circle at 20% 50%, 
    rgba(139, 126, 232, 0.15) 0%, 
    transparent 50%
)
```

### 버튼 스타일
```css
/* Primary 버튼 */
background: #8b7ee8
hover: translateY(-2px) + glow

/* Random 버튼 */
gradient: #ec4899 → #8b5cf6
hover: 상승 + 핑크-보라 효과

/* Reset 버튼 */
background: #2d2d2d
border: 1px solid #3d3d3d
hover: 밝아지기 효과
```

### 카드 효과
```css
background: rgba(30, 30, 30, 0.7)
backdrop-filter: blur(20px)
border: 1px solid rgba(255, 255, 255, 0.1)
box-shadow: 0 8px 32px rgba(0, 0, 0, 0.6)
```
